My Game Mods

Wildstar - Advanced Communicators

A Wildstar addon that extends the native CommDisplay addon by Carbine to give it more functionality and player to player communication via the communicators UI

LuaPlatform: Windows

Game: Wildstar

The Sims 3 - Community Server

The Sims 3 Online mod allows the creation of 20 player community town servers. Allowing the players to experience a multiplayer live town in real time.

This mod is currently in private testing.

C#Platform: Windows

Game: The Sims 3

Eco - Discord Access Mod

This mod provides Discord role based access to Eco servers. Allowing for more automatic control of who can access your community Eco servers. This mod can also be bundled with the native Discord Twitch integration and the Twitch Subscriber role to create subscriber only Eco servers.

This mod was originally created to demostrate the usage of the new IUserAuthorizer interface in Eco 9.5+. Any long term support or reliability cannot be guaranteed.

Note: This mod is currently dependant on an experimental ModKit api change for Eco 9.5+

C#Platform: WindowsPlatform: MacOSPlatform: Linux

Game: Eco

Eco - No More Books Mod

Tired of players getting a book from the research table just to hand scrolls out to everyone? Want to make your server more challenging? This mod will not allow books to be made on the research table, only scrolls, thus stopping players from making too many scrolls and giving them away.

Game: Eco
